**问题**: 多个线程同时访问共享数据

**Rust 方案**:
```rust

#### 4. **常见陷阱收集**

```markdown
# Rust 陷阱博物馆

## 陷阱 #1: 悬垂引用

```rust
fn dangling_reference() -> &i32 {
    let x = 5;
    &x  // ❌ x 在函数结束时被销毁
}
```

**真实案例**: 某创业者在生产环境遇到此 bug，导致随机崩溃

**修复方案**:
```rust
fn return_owned() -> i32 {
    let x = 5;
    x  // ✅ 返回所有权
}
```

**教训**: Rust 编译器阻止了你写出这个 bug
```
