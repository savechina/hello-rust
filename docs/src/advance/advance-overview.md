# 高级进阶 (Advance)

## 📖 学习内容概览

恭喜你完成了 **[基础入门](../basic/basic-overview.md)**！现在你已经掌握了 Rust 的核心概念。在**高级进阶**部分，你将学习如何使用 Rust 生态系统中的强大工具和库，构建真正的应用程序。

---

## 🎯 你将学到什么

完成本部分学习后，你将能够：

1. **异步编程** - 使用 Tokio 运行时和 async/await 语法
2. **数据库操作** - 使用 SQLx 和 Diesel 进行数据库编程
3. **Web 开发** - 使用 Axum 和 Hyper 构建 Web 服务
4. **数据处理** - 序列化/反序列化 JSON、CSV 等格式
5. **系统编程** - 文件操作、进程管理、系统信息
6. **测试与模拟** - 编写单元测试和集成测试

## 🔗 前置要求

**必须完成**:

- ✅ 理解所有权和借用
- ✅ 熟悉结构体、枚举、特征

**建议具备**:

## 📈 学习路径

```
异步编程 → 数据库 → Web 框架 → 序列化 → 文件操作 → 系统编程 → 测试与模拟
```

**推荐学习顺序**:

1. **核心基础** (必须先学):
   - 异步编程 → 数据库 → Web 框架

2. **数据处理** (按需学习):
   - JSON → CSV → 零拷贝序列化

3. **系统编程** (进阶):
   - 文件操作 → 进程管理 → 系统信息

4. **高级主题** (深入):
   - 宏编程 → MIO → 零拷贝
